# WsPiDroid

WsPiDroid est un Web-Service à installer sur un Raspberry Pi. Le but est de permettre à un développeur de faire interaction son application avec les ports GPIO de ce Raspberry Pi, et cela, quelque soit le langage de développement utilisé, dès lors qu'il est capable d'interagir avec un Web-Service de type SOAP.

WsPiDroid est composé d'un ensemble de script PHP, dont la librairie NUSaop, à déposer dans un répertoire Web du Raspberry Pi à interoger.
